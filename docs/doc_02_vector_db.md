# 🧠 02. 本地向量检索外脑 (Vector Retrieval)

## 🎯 模块简介
本模块搭建了一个轻量级的本地向量数据库，用于存储和高速检索企业内部知识（如 SOP、历史话术、供应商报价等）。它使系统具备了“语义理解”与“记忆”能力。

## 🛠️ 核心技术栈
* **向量数据库**: ChromaDB
* **Embedding 模型**: BAAI/bge-small-zh-v1.5 (国产顶级多语言特征提取模型)

## 🚀 运行前提
1. 首次运行会自动从 HuggingFace 下载 BGE 模型权重（约 90MB），请确保网络通畅。
2. 依赖库已安装：`chromadb`, `sentence-transformers`

## 💡 核心机制
1. **语义降维**: 将中文自然语言转化为高维数学向量系坐标。
2. **空间计算**: 采用相似度算法（如余弦相似度），精准匹配与用户口语化提问（甚至黑话）最接近的历史业务档案。
3. **标签过滤**: 支持通过 `metadatas` 给知识点打标签，实现精准范围控制。

## 🖥️ 使用示例
当传入类似“机子变砖头了咋办”的模糊查询时，系统会瞬间返回匹配度最高的标准处理 SOP，如“请指导买家同时长按电源键强制重启...”。
