# 🛒 Amazon Review RAG System (跨境电商智能客服引擎)

本项目是一个基于大语言模型 (LLM) 与向量检索技术 (ChromaDB) 构建的自动化数据清洗与智能客服回复系统。

## 🌟 核心功能
1. **非结构化数据清洗**：利用 LangChain + Pydantic 将冗杂差评转化为精准 JSON 数据。
2. **本地知识库检索**：搭载 BGE 多语言模型，实现业务 SOP 的毫秒级语义匹配。
3. **高情商自动回复**：大模型结合 SOP 护栏，自动生成专业、合规的安抚与赔偿邮件。

* `/src` - 核心业务代码与测试脚本
